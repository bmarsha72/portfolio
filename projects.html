Hate State Navigate

click on photo

Hate State Navigateclose

Created the week following North Carolina's foray into bathroom law idiocy, this project's aim was to allow businesses to register their bathrooms as transgender accessible and offer a pin-dropping map search function for users. I came up with the idea, having watched the story unfold on the news feed while riding the El in the morning. I also assisted with the Google map API, and the backend routes. My favorite part, aside from learning from my group, is the graphic on the front with the three bathroom doors. With more strange bathroom laws cropping up all over the country, this project may get revitalized in the near future, although it's not currently live.

Mu/Chart Generator

Mu/Chart Generatorclose

Inspired by Essential Charts, Mu/Chart Generator is an app that allows users to build and share music album flowcharts. This generator is an easy way to visually categorize your musical tastes. I loved working in this group, as I got to learn about React drag and drop components. My involvement was with the planning, Git management, some assistance with views, research on a API display issue, and the tag line "Turn On, Tune In, and Chart Out".

Trump Libs

link to site

Trump Libsclose

My first solo project in the WDI program, Trump Libs is an API created using HTML5 audio tags nested inside callback functions to make very funny sentences of certain republican candidates. These mp3 clips were pulled from YouTube sources in the public domain and are from a combination of interviews and public speaking engagments. This project was a big hit with the class, and I'm planning to turn it into a mobile app using Cordova. The biggest challenge on this project was the sheer amount of audio I had to plow through to get to the good clips. The javascript itself was also a bit finicky.
