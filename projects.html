Mu/Chart Generatorclose

Inspired by Essential Charts, Mu/Chart Generator is an app that allows users to build and share music album flowcharts. This generator is an easy way to visually categorize your musical tastes. I loved working in this group, as I got to learn about React drag and drop components. My involvement was with the planning, Git management, some assistance with views, research on a API display issue, and the tag line "Turn On, Tune In, and Chart Out".

Trump Libs

link to site

Trump Libsclose

My first solo project in the WDI program, Trump Libs was created using HTML5 audio tag callbacks to make funny sentences of a certain republican candidate. These mp3 clips were pulled from YouTube sources in the public domain and are from a combination of interviews and public speaking engagments. Unfortunately, the audio callbacks only work on desktop devices at the moment. The biggest challenge on this project was the sheer amount of audio I had to plow through to get to the good clips, and matching the amplitude of the clips. And, of course, I didn't find out about javascript promises until AFTER I was finished. Typical.

My contributions to this company site include, but aren't limited to, header and footer layout, common needs pages including stock photo positioning and breakpoints in addition to text content and editing, payment modal design, responsive adaptations for buttons, etc. Some examples of server side sub-projects include creating an interface for interacting with SMS messages with current and potential clients, creating a table to store browser info from visitors, attaching an AJAX call to a submit button and processing queries to multiple tables to return information about the submitted form, creating automatic response emails based on query information, etc.
